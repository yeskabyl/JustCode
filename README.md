# JustCode AI Agent (Telegram Bot)

Универсальный ИИ-агент для студентов и преподавателей, разработанный в рамках финального проекта JustCode.

## Основные возможности

### Для Студентов:
- **AI-Репетитор**: Пошаговое объяснение сложных тем.
- **Проверка ДЗ**: Анализ решений по фото или тексту с помощью GPT-4o.
- **Тренажер**: Генерация задач для закрепления знаний.
- **Шпаргалки**: Создание компактных конспектов по темам.

### Для Преподавателей:
- **Расписание**: Автоматизация планирования уроков.
- **Проверка работ**: Быстрая проверка и оценка работ учеников.
- **Генерация материалов**: Создание планов уроков, тестов и заданий.
- **Аналитика**: Экспорт успеваемости в Excel.
- **Презентации**: Разработка структуры и дизайна учебных презентаций.

## Установка и запуск

1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создайте файл `.env` на основе `.env.example` и добавьте ваши ключи:
   - `BOT_TOKEN`: Токен от BotFather.
   - `OPENAI_API_KEY`: Ключ OpenAI.
4. Запустите бота:
   ```bash
   python main.py
   ```

## Технологии
- **aiogram**: Современный асинхронный фреймворк для Telegram.
- **OpenAI API**: GPT-4o для анализа текста и изображений.
- **Pandas**: Обработка данных и генерация отчетов.
